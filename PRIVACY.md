# Privacy Policy

Last updated: March 20, 2026

LineFlow stores operational dashboard data in your browser local storage by default.

## What data is stored

- Prep items
- Inventory items
- 86'd items
- Shift notes
- Activity log entries

## Where data is stored

- Data is stored locally on the device/browser profile in use.
- Data is not transmitted to a backend service by this application.

## Backup files

- Users can export data to a JSON backup file.
- Importing a backup replaces current in-app state with imported data.

## Your controls

- Clear browser storage to remove local data.
- Use in-app clear actions to remove records.
- Export backups before clearing data if recovery is needed.

## Contact

For support or privacy questions, open an issue in the project repository.
